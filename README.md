# Herbal Wellness App

This app provides information and resources related to herbal wellness, including various herbs, their benefits, and how to use them safely.

## Features
- Herb Catalog
- Dosage Information
- User Authentication
- Community Resources

## Getting Started
1. Clone the repository
2. Install dependencies using `npm install`
3. Start the application with `npm start`

## Technologies Used
- Node.js
- Express
- React
- MongoDB
